
**Les caractères speciaux
lorsqu' on veux ecrire des caratères specaux dans le text il faut le faire précéder d'un backslash.
	\\   : Anti-slash
	\`   : Apostrophe curve
	\*   : Asterisk
	\_   : Low dash / Underscore
	\{\} : Braces
	\[\] : Hooks
	\(\) : Parentheses
	\#   : Sharp
	\+   : Sign more
	\-   : Less sign / hyphen
	\!   : Exclamation dot
	\.   : Dot
